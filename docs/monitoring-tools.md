# ابزارهای گرافیکی مدیریت داکر و لاگ‌ها

آیا نیاز است؟ خیر، اما برای راحتی و دیباگ سریع بسیار مفید است.

## گزینه ۱: Portainer (مدیریت کامل + لاگ)
محبوب‌ترین ابزار مدیریت داکر. به شما امکان می‌دهد کانتینرها، ایمیج‌ها و شبکه‌ها را مدیریت کنید و لاگ‌ها را ببینید.

### نصب روی سرور
یک فایل `docker-compose.portainer.yml` جداگانه بسازید یا مستقیماً دستور زیر را اجرا کنید:

```bash
docker run -d -p 8000:8000 -p 9443:9443 --name portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer-ce:latest
```

### دسترسی
- آدرس: `https://89.42.199.69:9443`
- نکته: چون گواهی SSL معتبر ندارید، مرورگر اخطار می‌دهد (Advanced -> Proceed را بزنید).
- بار اول یک یوزر و پسورد قوی بسازید.

### امنیت
- پورت 9443 را در فایروال باز کنید: `ufw allow 9443`
- پسورد قوی بگذارید.

## گزینه ۲: Dozzle (فقط مشاهده لاگ‌ها)
اگر فقط می‌خواهید لاگ‌ها را راحت ببینید و نیازی به مدیریت ندارید. بسیار سبک و سریع.

### نصب
```bash
docker run -d --name dozzle -p 8888:8080 -v /var/run/docker.sock:/var/run/docker.sock amir20/dozzle:latest
```

### دسترسی
- آدرس: `http://89.42.199.69:8888`
- لیست تمام کانتینرها سمت چپ است، روی هر کدام کلیک کنید لاگ زنده را می‌بینید.

---

## گزینه ۳: VS Code Remote - SSH (پیشنهاد برنامه‌نویس‌ها)
بدون نصب ابزار اضافی روی سرور، محیط سرور را به VS Code خودتان می‌آورید.

1. افزونه **Remote - SSH** را در VS Code نصب کنید.
2. دکمه `< >` (گوشه پایین چپ) را بزنید و **Connect to Host** را انتخاب کنید.
3. دستور `ssh root@89.42.199.69` را وارد کنید.
4. پسورد سرور را بزنید.
5. حالا فایل‌های سرور را در فایل اکسپلورر می‌بینید و ترمینال VS Code شما ترمینال سرور است.
6. افزونه **Docker** را روی VS Code (در حالت ریموت) نصب کنید تا کانتینرها و لاگ‌ها را گرافیکی ببینید.
